** 配置

Git 是分布式版本控制系统，需要每台电脑自报家门：你的名字和 Email 地址。我们使用 =git config= 来完成这件事情，如下：

#+BEGIN_SRC sh -n
→ git config --global user.name "Jack Liu"
→ git config --global user.email "loveminimal@163.com"
#+END_SRC
